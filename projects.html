Link-It

Live site

Project Overview

Link-It is a LinkTree clone, that allows users to create personalized landing pages where they can manage all their online links and social media pages. Other features include user authentication, user profile functionality, allowing users to create and manage their profiles(profile images are also supported). It also includes a forgotten password feature that allows users to easily reset their old password and regain access to their account. Link-It is a good choice for anyone looking to streamline and organize their online presence. HTML, SCSS, JavaScript, Django, PostgreSQL
